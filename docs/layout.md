# Cascadify Layout

```
<div class="layout">
    <div header>My tiny site.org</div>
    <div nav>
        <a href="#>Home</a>
        <a href="#>About</a>
        <a href="#>Billings</a>
        <a href="#>Legal</a>
    </div>
    <div body>Body here...</div>
    <div footer>Footer here</div>  
</div>  
```

Cascadify recognizes that it is a layout. Now Cascadify knows that the layouts can contain: the header, the navigation (nav), the body and the footer.

Cascadify only works with classes when you want to add a component. Cascadify knows that, for example, a card can contain an image and some text and buttons.
